# Dr. Lukas Berger — Tech-Lead bei mittelstaendischem Maschinenbauer

## Demografie
- 41 Jahre, promovierter Informatiker, 12 Jahre Berufserfahrung
- CTO bei einem 350-Mitarbeiter-Maschinenbauer in Sueddeutschland
- Verantwortet IT-Architektur, Digitalisierungs-Roadmap und Datenarchitektur

## OCEAN-Profil
- **Openness:** 0.75 — neugierig auf neue Tools, evaluiert MCP-Stack aktiv
- **Conscientiousness:** 0.85 — sehr strukturiert, dokumentiert alles
- **Extraversion:** 0.45 — eher introvertiert, fokussiert auf Inhalte
- **Agreeableness:** 0.55 — direkt, aber kollegial
- **Neuroticism:** 0.30 — souveraen, geringe Frustrationstoleranz bei Tools

## Goals
1. ERD fuer SAP-Migration dokumentieren — 47 Tabellen, 130+ Relations
2. Prozess-Flows fuer Auftragsabwicklung modellieren (BPMN)
3. System-Landscape (C4) fuer ISO 27001-Audit erstellen
4. Alles soll **versionierbar** sein (Git-Diff-friendly)
5. Agent-driven Editing — Claude Code soll von Beschreibungen Diagramme bauen

## Pain Points
- Hat schon Lucidchart, draw.io, Visio probiert — alle nicht agent-native
- Keine Zeit fuer manuelle Diagramm-Pflege bei 47 Tabellen
- Will Code als Source-of-Truth (DBML / JSON) statt Klick-Modelle
- Erwartet Shortcuts wie in IDEs (VSCode, JetBrains)
- Bevorzugt Keyboard-Workflow vor Mouse

## Knowledge Level
- **Sehr hoch** bei: SQL, ERDs, BPMN, MCP, JSON-Schemas, Git
- **Mittel** bei: React, Frontend-UX
- **Niedrig** bei: Programmatic SEO

## Test-Verhalten
- Wird ERST den Code-Pfad probieren (Cmd+/, dann DBML editieren)
- Wechselt zwischen Klick und Shortcut je nach Effizienz
- Erwartet, dass `set_dbml` MCP-Tool auch im UI direkt verfuegbar ist
- Wird auf fehlende Landscape-Sidebar-Eintraege sofort frustriert reagieren
- Misst Zeit pro Aufgabe innerlich

## Erwartete Tasks (Connextra)
- Als Tech-Lead **moechte ich** ein ERD aus DBML-Code importieren, **damit** ich mein Schema versionieren kann.
- Als Tech-Lead **moechte ich** zwischen ERD, BPMN und System-Landscape per Tab/Shortcut wechseln, **damit** mein Architektur-Workspace komplett bleibt.
- Als Tech-Lead **moechte ich** per Cmd+K alle Editor-Aktionen finden, **damit** ich die Maus nicht brauche.
- Als Tech-Lead **moechte ich** ein Bundle exportieren (DBML + Mermaid + SVG + Positionen), **damit** ich es Stakeholdern zeigen kann.
- Als Tech-Lead **moechte ich** das System-Landscape per JSON-Code editieren, **damit** ich grosse Aenderungen schnell durchfuehren kann.
